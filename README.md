# Q&A
실습 진행 도중에 생긴 오류 해결에 어려움을 겪는다면 [이슈](https://github.com/shinsunyoung/springboot-developer/issues)를 등록해주시거나 [오픈카톡](https://open.kakao.com/o/gE422Qtf)으로 문의 주시면 최대한 빠르게 답변 드릴 수 있도록 하겠습니다.
